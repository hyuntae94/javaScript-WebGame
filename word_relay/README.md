<h1 align="middle">
    끝말잇기
</h1>

**🚦구현순서**
1. 몇 명이 참가할지 선택한다.
    - `input태그` 와 `prompt함수중` 직관적인 `prompt함수` 사용
2. 단어를 입력한다.
    - `첫번째 참가자O` -> 입력한 단어가 제시어가 된다.
    - `첫번째 참가자X` -> 4번으로 이동
3. 다음 참가자가 단어를 말한다.
4. 두 단어가 연결되는지 확인한다. 
    - `연결O` -> 3번으로 이동한다.
    - `연결X` -> 틀렸다고 표시하고 프로그램 종료한다.  

**📌 규칙**

1) 태그를 저장하는 변수명은 `$`로 시작.
